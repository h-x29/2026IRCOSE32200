* TCP review
* TCP data structures
* Sending TCP segments
* Receiving TCP segments



\## TCP vs UDP



TCP provides reliable communication, while UDP provides best-effort delivery.



TCP is more complex because it supports:



* Connection-oriented communication
* Reliable delivery
* In-order packet delivery
* Flow control
* Congestion control



\## Key Idea



TCP is not only about sending packets.

It also manages connection states, retransmission, packet ordering, receiver buffer limits, and network congestion.
